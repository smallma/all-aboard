# Avatar Assets

The human avatar images were generated from DiceBear's Avataaars style.

- Source: https://www.dicebear.com/styles/avataaars/
- API: https://api.dicebear.com/9.x/avataaars/svg
- Design license: Free for personal and commercial use
- DiceBear code license: MIT

The animal avatar images are original generated vector assets created for this project.

Generated files:

- `avatar-01.png` through `avatar-24.png`
- `avatar-25.png` through `avatar-64.png`
- `asian-01.png` through `asian-12.png`
- `animal-01.png` through `animal-12.png`
- `manifest.json`
